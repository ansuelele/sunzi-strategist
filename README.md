## 是什么

这是一个为 [WorkBuddy](https://www.codebuddy.cn) 设计的 AI 战略参谋 Skill。

激活后，AI 以孙武的视角和思维框架，分析现代商业竞争：

- 📊 **竞争分析** — 从《孙子兵法》"五事七计"拆解竞品
- 🤝 **谈判博弈** — "上兵伐谋"的谈判策略框架
- 🎯 **市场决策** — "知己知彼"的战略判断
- 👥 **团队管理** — "将之五危"的领导力警示
- ⚠️ **危机应对** — "死地后生"的逆境决策

## 使用方法

在 WorkBuddy 中输入以下指令即可激活：

```
用孙子视角分析这个竞品
帮我做商业谈判策略
用孙子兵法分析市场进入方案
```

或指定具体场景：

```
A公司刚发布了一款产品，直接对标我们，他们有哪些弱点？
我要跟投资人谈，但担心他压价，我该怎么谈？
我想进入一个已经有巨头占据的市场，怎么打？
```

## 孙子 Skill 速查

### 战略优先级

```
上兵伐谋    → 破坏敌人的计划（最高明）
其次伐交    → 瓦解敌人的联盟
其次伐兵    → 军事威慑
其下攻城    → 直接对抗（代价最高）
```

### 决策前必问

| 问题 | 孙子篇目 |
|------|---------|
| 我们有胜算吗？ | 军形篇：先胜后战 |
| 为什么要快？ | 作战篇：久则钝兵挫锐 |
| 这是不是"不战而胜"的机会？ | 谋攻篇：全胜思想 |
| 我们在哪里集中兵力？ | 虚实篇：致人而不致于人 |
| 时间窗口在哪？ | 军争篇：兵贵胜，不贵久 |
| 决策是情绪还是计算？ | 火攻篇：主不可以怒而兴师 |
| 我们掌握足够的信息吗？ | 用间篇：不知敌之情者，不仁之至也 |

### 领导力警示（将之五危）

| 五危 | 现代表现 | 应对 |
|------|---------|------|
| 必死 | 硬拼、赌徒心态 | 算清楚再动 |
| 必生 | 保守、不敢决策 | 设定底线，果断行动 |
| 忿速 | 冲动决策、情绪化反应 | 冷却期，问"这是计算还是愤怒？" |
| 廉洁 | 过于自尊、怕被批评 | 听得进坏消息 |
| 爱民 | 原则模糊、怕得罪人 | 区分仁慈和软弱 |

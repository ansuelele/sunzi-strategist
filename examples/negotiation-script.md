# 示例：谈判策略 — 投资人压价谈判

> 使用 sunzi-strategist Skill 生成的投资人谈判策略示例。

---

## 输入

```
我要跟投资人谈A轮融资，但担心他压价。
我该怎么准备？谈判过程中有哪些策略可以用？
```

---

### 【庙算】谈判前该准备什么

> "夫未战而庙算胜者，得算多也。"

**你的"五事"（筹码评估）：**

| 五事 | 你的现状 | 评估 |
|------|---------|------|
| 道 | 愿景清晰，但团队磨合期短 | ★★★☆☆ |
| 天 | 赛道是窗口期，但已有竞品 | ★★★☆☆ |
| 地 | 用户数据好，但规模小 | ★★★☆☆ |

| 法 | 股权结构清晰，激励机制 | ★★★★☆ |

**结论：** 你有筹码，但不够碾压。庙算"小胜"。

**投资人的"五事"（了解对手）：**

| 五事 | 投资人心理 |
|------|----------|
| 道 | 他这一期基金的战略是什么？偏好什么阶段？ |
| 天 | 他现在手里有几个项目在谈？ |
| 地 | 他的LP结构，谁给他的压力？ |
| 将 | 谈判的人是什么风格？（直接型/情感型/拖延型） |
| 法 | 他的决策流程是什么？一个人能拍板吗？ |

**核心问题：** "知彼知己"——你知道他要什么吗？

---

### 【伐谋】如何不战而屈人之兵

> "不战而屈人之兵，善之善者也。"

**投资人压价的底层逻辑：**
- 他想用更低的价格拿到更多股份
- 他不确定你是不是最好的选择

**"伐谋"策略：让他"主动想投你"**

**策略一：制造稀缺性**

孙武曰："故形人而我无形，则我专而敌分。"

→ 让他知道你有其他投资人在谈（即使没有，也要让他感觉"不投就没了"）

→ 话术："我们这轮大概开放2-3个领投名额，目前有两家FA在帮我们筛选。"

**策略二：重新定义"估值"**

→ 不是"我要多少钱"，而是"你投了这笔钱，三年后值多少"

→ 聊行业天花板，聊你的市场地位，聊退出路径

→ 让他算，算完之后发现"便宜"。

**策略三：攻其软肋（用"五危"分析投资人）**

| 五危类型 | 投资人表现 | 应对 |
|---------|----------|------|
| 必死 | 急于投出这一期，不想错过好项目 | 给他"错过就是损失"的压力 |
| 必生 | 担心投错被LP问责 | 给他足够的安全垫和退路 |
| 忿速 | 谈判中被激怒，直接表态 | 不要激怒他，保持冷静 |
| 廉洁 | 过于在意条款的"公平性" | 把条款设计得"看起来公平" |
| 爱民 | 对创始人有情感认同 | 建立私人关系，让他"押注人" |

---

### 【伐交】可以联合谁孤立他

> "其次伐交"——瓦解对方的联盟。

**你有其他投资人备选吗？**

- 如果有：他压价你就走，有退路才能谈判
- 如果没有：优先建立备选，哪怕是小额TS

**谈判前的联盟策略：**
- 同时推进2-3个投资人的沟通
- 让A投资人知道B投资人在谈
- 这不是威胁，这是"伐交"

---

### 【伐兵】万不得已怎么办

> "其次伐兵"——必要时，展示你的底线。

**什么时候亮底牌？**

"先为不可胜，以待敌之可胜。"

→ 你先确保自己立于不败之地（有其他融资选项），再亮底牌。

**底牌是什么？**

1. 明确告知：这是你能接受的最低估值，低于这个数字你不拿
2. 备选方案：已经有其他投资人在做DD
3. 真实原因：为什么这个估值对你重要（给一个合理的解释）

**什么时候该离开谈判桌？**

"合于利而动，不合于利而止。"

→ 条款触及底线，立刻走。不要为了"拿到钱"而拿不该拿的钱。

---

### 【谈判全程框架】

```
Phase 1: 开场 — 建立势能
- 不要先开口谈估值，让他先出价
- 先聊行业，聊格局，聊你看到的他没看到的东西

Phase 2: 伐谋 — 重新定义估值
- 把估值问题转化为"你投了，三年后值多少"的问题
- 展示你的稀缺性和选择权

Phase 3: 伐交 — 展示联盟
- "我们在跟几家也在聊"，不需要说名字
- 让他有竞争压力

Phase 4: 伐兵 — 明确底线
- 给出你能接受的最低条件
- 说明原因（不是情绪，是逻辑）

Phase 5: 收尾 — "成交或不成交"
- 孙武曰："合于利而动，不合于利而止。"
- 不要恋战，不要妥协超过底线
```

---

### 【孙武一句话】

> "知彼知己，百战不殆。"

谈判的核心不是压价，是"了解"——了解他，了解自己，了解为什么他想投你或压你价。

**当你真正"知彼"了，你会发现很多压价，其实是可以化解的。**
